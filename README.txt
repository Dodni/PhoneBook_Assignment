Üdv, ez a program egy telefonkönyv adatbázis.
Az indító file a bin azután debug majd net5.0-windows mappában a WPFEFCMF_PhoneBook.exe 
Elsőnek egy bejelentkező ablak fogad majd a John 12345 vagy Jane 12345 felhasználóval tudunk belpni.
Ha beléptünk rögtön fogad minket az adatbázis, ahol kitudjuk listázni az összes várost és vagy az összes személyt, illetve a személyek adatit. 
Tudunk új telefonszámot meg várost hozzáadni vagy már meglévőket szerkeszteni.


Ezt az egészet egy iskolai kötelező beadandó miatt készítettem, úgy gondolom rengeteget tanultam belőle, rettentő hasznos volt.
